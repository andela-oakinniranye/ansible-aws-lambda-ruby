Ansible AWS Lambda Ruby

This is an Ansible role I created to play around with AWS Lambda and package up a Ruby script to be executed by a Lambda function. Traveling Ruby is used to include the Ruby runtime in the package.

The Ansible AWS Lambda module is not yet included in core, so for now it must be present in any playbook that uses this role. Here is an example playbook I used to test this role.

I sure hope AWS Lambda supports Ruby natively soon!
